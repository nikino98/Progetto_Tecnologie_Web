# Progetto_Tecnologie_Web: Pizzeria Ristorante
Progetto di un sito per la gestione di una Pizzeria Ristorante.
Clonare la repository nella vostra directory.

<h3>Prerequisiti</h3>
----------------------
<ul>
  <li>python </li>
  <li>pipenv</li>
  <p>Installazione: <code>pip3 install pipenv</code></p>
</ul>
